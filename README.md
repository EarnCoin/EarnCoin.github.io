# 從零開始
比特幣是一種用去中心化、全球通用、不需第三方機構或個人，基於區塊鏈作為支付技術的電子加密貨幣。比特幣由中本聰(Satoshi)於2009年1月3日，基於無國界的對等網路，用共識主動性開源軟體發明創立。比特幣也是目前知名度與市場總值最高的加密貨幣。 任何人皆可參與比特幣活動，可以通過稱為挖礦的電腦運算來發行。
